## $C^2hinese$ 简介
基于C语言，我们开发了一套完全支持中文并兼容英文的编程语言CChinese, 简称$C^2$，并基于Java开发了一个简单的编译器。 相较于现有中文编程语言，$C^2$ 有如下亮点：

* 中文变量名与关键字
* 中文操作符与描述
* 全面支持中文字符，避免了中英文输入法来回切换
* 兼容性：完美兼容中英两种语言

详细介绍可参见CChinese设计报告
